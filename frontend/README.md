* React + Vite + TypeScript + npm
* Stylizowanie i gotowe komponenty: TailwindCSS v4 + shadcn
* Routing: ReactRouter

Uruchamianie:
    1. W folderze /frontend wywołaj: npm install
    2. Uruchom serwer frontendowy: npm run dev
    3. Aplikacja będzie dostępna pod http://localhost:5173

Proponowane biblioteki na przyszłość:
* TanStackQuery (+axios)
* Zod
* Zustand
* date-fns
* jwt-decode
* Lucide React
